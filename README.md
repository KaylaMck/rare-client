# Rare: The Publishing Platform for the Discerning Writer

A multi-user blogging platform built with React 18, React Router v6, and Bulma CSS.

## Getting Started

1. Make sure the [API server](https://github.com/NSS-Workshops/rare-api) is running on port 8088
2. Install dependencies: `npm install`
3. Start the dev server: `npm start`
4. Open http://localhost:3000
